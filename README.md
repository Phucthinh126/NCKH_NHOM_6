# GNN - Bayesian Graph Attention Network

## Giới thiệu

Dự án nghiên cứu và thực nghiệm các mô hình Graph Neural Network (GNN), bao gồm:

- GAT (Graph Attention Network)
- Bayesian GAT
- Bayesian Approximate GAT

Các notebook được viết bằng **Python** và chạy trên **Jupyter Notebook**, **Google Colab** hoặc **Visual Studio Code**.
